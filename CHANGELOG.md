## [0.0.5] - 2025-02-07

### Added

- `chaud -a` to show all available devices.
- Support for device numbers (in addition to names) in config.
- Unit tests.
- Automatic JSON Schema validation

### Changed

- `deviceId` changed to `device` in configuration schema.
